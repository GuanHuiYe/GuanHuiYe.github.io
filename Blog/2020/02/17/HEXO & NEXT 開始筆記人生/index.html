設定網址

如果你的網站不是放在最上層的話 例：http://xxx.xxx.com/xxx/…
那麼就要特別設定 url 及 root
我們來看到 #URL 的部分

url:的部分很簡單就是你網站放置的網址
那root：的部分就是放置你網站起始的點
例：
我的網站是 http://www.abc.com/
但我的部落格放在 http://www.abc.com/blog
那就會長這樣

設定文章圖片

如果你的文章要用圖片，但是有不想要將照片丟在雲端
那麼這個設定一定能幫助到你
到 # Writing 的區塊將

改成

post_asset_folder: true

這樣在新增文章時會自動產生對應的資料夾，將照片放入就能夠使用圖片了喔～
