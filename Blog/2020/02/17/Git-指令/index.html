Git 基本指令

Git Add

語法 說明
git add . 將所有檔案放置佔存庫
git add 檔案位置 將特定檔案放置佔存庫

Git Commit

語法 說明
git commit -m Commit 檔案

Git Branch

語法 說明
git branch 查看所有分支
git branch 分支名稱 新增分支
git branch -d 分支名稱 刪除分支

Git Remote

語法 說明
git remote 查看遠端位置
git remote -v 查看詳細遠端位置
git remote add ip/*.git 建立遠端 Git 連線
git rm 遠端名稱 刪除 遠端位置

Git 顯示指令

語法 說明
git log 查看 Git 歷史紀錄
git log –graph 查看 Git 歷史紀錄 (更詳細)
gitk –all 視窗化查看 Git 歷史紀錄
